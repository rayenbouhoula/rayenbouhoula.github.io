## üé® Customization

### Theme Colors

Edit `src/index.css` to customize the color scheme:

**Dark Mode:**
- Primary Background: `#0a0a0a`
- Accent Color: `#00d9ff`

**Light Mode:**
- Primary Background: `#ffffff`
- Accent Color: `#0099cc`

### Personal Information

Update your information in `src/config.js`:
- Name, title, description
- Social media links
- Contact information

### CV File

Replace the placeholder CV at `public/cv/Rayen_Bouhoula_CV.pdf` with your actual CV.

### Sound Effects

Add your custom sound files to `public/sounds/`:
- `click.mp3`
- `whoosh.mp3`
- `success.mp3`
- `toggle.mp3`
- `error.mp3`
